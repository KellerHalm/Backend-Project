# API написанный на NodeJS с CRUD-операциями
### Присутствуют операции регистрация и авторизация пользователей
### Создание, получение, изменение и удаление товара

```bash
Клонируйте репозиторий
git clone https://github.com/KellerHalm/Backend-Project.git

# Перейдите в папку проекта
cd Backend-Project

# Установите зависимости
npm install

# Создайте .env файл

# Запуск сервера
npm run start
```
